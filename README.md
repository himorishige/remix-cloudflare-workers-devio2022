## RemixとCloudflare Workersでつくるサービス指向アーキテクチャ

Cloudflare Workers上でWorkers Service Bindingsを利用し複数のRemixアプリケーションをマイクロサービス的に利用するアイデアについてお話します。  
RemixのほかサーバーサイドにはフレームワークHonoを、認証基盤にはSupabaseを利用した構築例となっています。
